
Ibn Al‑Nasr Battle – Light Version (Starter Package)

This package contains:
• Basic Unity scripts
• Game design document
• Map planning for Nasiriyah
• Feature list for vehicles, drones, and buildings

To use:
1. Install Unity Hub
2. Create a new 3D project
3. Copy the Assets folder from this package into your Unity project
4. Start editing the scene and adding models
